## Setup

1. Clone repo: `git clone ...`
2. Setup MySQL DB, run schema in `.sql`
3. Copy `.env.example` to `.env` & fill credentials
4. Install dependencies:
pip install -r requirements.txt
5. Run server:
python app.py
6. Visit `http://localhost:5000` in browser
